Git Workflow Guide

Interactive cheat sheet for Git version control

Git Basics

git init

Initialize a new Git repository in the current directory.

git init

git status

Show the state of your working directory — untracked, modified, or staged files.

git status

git add

Stage files for the next commit. Use . to stage all changes.

git add <file>

git commit

Save staged changes as a snapshot with a descriptive message.

git commit -m "Your message here"

git log

View commit history. Use --oneline for a compact view.

git log --oneline
