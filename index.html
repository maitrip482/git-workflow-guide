Git Workflow Guide

Interactive cheat sheet for Git version control

Git Basics

git init

Initialize a new Git repository in the current directory.

git init

git status

Show the state of your working directory — untracked, modified, or staged files.

git status

git add

Stage files for the next commit. Use . to stage all changes.

git add <file>

git commit

Save staged changes as a snapshot with a descriptive message.

git commit -m "Your message here"

git log

View commit history. Use --oneline for a compact view.

git log --oneline

Branching & Merging

git branch

List all local branches. The current branch is marked with *.

git branch

git switch -c

Create a new branch and switch to it immediately.

git switch -c feature/my-feature

git switch

Switch to an existing branch. Your files update to match that branch's state.

git switch main

git merge

Merge another branch into your current branch. Use --no-ff to preserve branch history.

git merge feature/my-feature

git branch -d

Delete a branch after it has been merged. Use -D to force-delete unmerged branches.

git branch -d feature/my-feature

Remote Operations

git remote add

Link your local repository to a remote (e.g., GitHub). origin is the conventional name.

git remote add origin https://github.com/user/repo.git

git push

Upload local commits to the remote. Use -u on first push to set tracking.

git push -u origin main

git pull

Download and merge remote changes into your current branch (fetch + merge).

git pull origin main

git fetch

Download remote changes without merging. Safe way to check what's new.

git fetch origin

git clone

Copy an entire remote repository to your local machine with full history.

git clone https://github.com/user/repo.git

Undoing Changes

git restore

Discard uncommitted changes in a file. Reverts to last committed state.

git restore <file>

git restore --staged

Unstage a file without losing your changes. Moves it back to working directory.

git restore --staged <file>

git revert

Create a new commit that undoes a previous commit. Safe for shared branches.

git revert HEAD

git reset

Move branch pointer backward. --soft keeps changes staged, --hard discards everything.

git reset --soft HEAD~1

git stash

Temporarily shelve uncommitted work. Use pop to restore later.

git stash && git stash pop
